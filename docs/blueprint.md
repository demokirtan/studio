# **App Name**: Noirfolio

## Core Features:

- Project Gallery: Display creative projects with images and videos in a visually appealing layout, drawing inspiration from futurethree.studio.
- About Section: Provide a narrative about the individual or studio, highlighting experience and philosophy.
- Contact Form: Allow visitors to send messages or inquiries through a simple contact form.
- Smooth Navigation: Implement smooth transitions and interactions for enhanced user experience, taking cues from futurethree.studio's design.

## Style Guidelines:

- Primary color: Dark gray (#333333) for the main elements, providing a neutral canvas.
- Background color: Almost black (#1A1A1A) for a stark, dark mode aesthetic.
- Accent color: Light gray (#CCCCCC) for subtle highlights and interactive elements, ensuring readability without being too stark.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text. 'Inter' has a neutral and modern appearance suitable for portfolio websites. 'Source Code Pro' (monospace) for computer code if required.
- Minimalist layout with ample spacing, inspired by futurethree.studio, focusing on showcasing the projects effectively.
- Subtle fade-in and transition effects when loading content or navigating between sections to create a smooth experience.
